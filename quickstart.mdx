# Quickstart

Get started with Sniffmail SDK in just a few lines of code.

## Basic Validation (Free)

Basic validation includes syntax checking, disposable email detection, and MX record verification. **No API key required.**

<Steps>
  <Step title="Import the library">
    ```typescript
    import { validateEmail } from 'sniffmail';
    ```
  </Step>
  
  <Step title="Validate an email">
    ```typescript
    const result = await validateEmail('someone@gmail.com');

    if (!result.valid) {
      console.log(`Invalid: ${result.reason}`);
      // Possible reasons: 'invalid_syntax', 'disposable', 'no_mx_records'
    }
    ```
  </Step>
</Steps>

### Complete Example

```typescript
import { validateEmail } from 'sniffmail';

const result = await validateEmail('someone@gmail.com');

console.log(result);
// {
//   email: 'someone@gmail.com',
//   valid: true,
//   reason: null,
//   disposable: false,
//   mx: true,
//   smtp: null,
//   cached: false
// }
```

<Note>
  Basic validation is **completely free** and doesn't require an API key. It checks syntax, disposable domains (26,000+ domains), and MX records.
</Note>

## Deep Mode (SMTP Verification)

Deep mode verifies that a mailbox actually exists by connecting to the mail server via SMTP. This catches fake emails like `fakeperson123@gmail.com`.

<Warning>
  Deep mode requires a Sniffmail API key. Get your free API key at [https://sniffmail.io](https://sniffmail.io)
</Warning>

<Steps>
  <Step title="Get your API key">
    Sign up at [https://sniffmail.io](https://sniffmail.io) and get your API key. The free tier includes 500 verifications per month.
  </Step>
  
  <Step title="Set your API key">
    You can set your API key using environment variables or the `configure()` function.

    <CodeGroup>

    ```bash Environment Variable
    export SNIFFMAIL_API_KEY=sniff_xxxxx
    ```

    ```typescript configure() Function
    import { configure } from 'sniffmail';

    configure({
      apiKey: process.env.SNIFFMAIL_API_KEY,
    });
    ```

    </CodeGroup>
  </Step>
  
  <Step title="Enable deep mode">
    ```typescript
    import { configure, validateEmail } from 'sniffmail';

    configure({
      apiKey: process.env.SNIFFMAIL_API_KEY,
    });

    const result = await validateEmail('someone@gmail.com', { deep: true });

    if (result.smtp?.is_reachable === 'invalid') {
      console.log('Mailbox does not exist!');
    }
    ```
  </Step>
</Steps>

### Complete Deep Mode Example

```typescript
import { configure, validateEmail } from 'sniffmail';

// Configure once at app startup
configure({
  apiKey: process.env.SNIFFMAIL_API_KEY,
});

const result = await validateEmail('someone@gmail.com', { deep: true });

console.log(result);
// {
//   email: 'someone@gmail.com',
//   valid: true,
//   reason: null,
//   disposable: false,
//   mx: true,
//   smtp: {
//     is_reachable: 'safe',
//     can_connect: true,
//     is_deliverable: true,
//     is_catch_all: false
//   },
//   cached: false
// }
```

## Understanding SMTP Results

When using deep mode, the `smtp` field provides detailed information about mailbox reachability:

| Status | Description |
|--------|-------------|
| `safe` | Email exists and is deliverable |
| `risky` | Email exists but might bounce (catch-all, full mailbox) |
| `invalid` | Email does not exist |
| `unknown` | Could not determine (timeout, blocked, etc.) |

```typescript
const result = await validateEmail('test@example.com', { deep: true });

if (result.smtp?.is_reachable === 'safe') {
  console.log('✓ Email is safe to send to');
} else if (result.smtp?.is_reachable === 'risky') {
  console.log('⚠ Email might bounce');
} else if (result.smtp?.is_reachable === 'invalid') {
  console.log('✗ Email does not exist');
}
```

## Validation Reasons

When an email is invalid, the `reason` field explains why:

```typescript
const result = await validateEmail('invalid-email', { deep: true });

switch (result.reason) {
  case 'invalid_syntax':
    console.log('Email format is invalid');
    break;
  case 'disposable':
    console.log('Domain is a known disposable email provider');
    break;
  case 'no_mx_records':
    console.log('Domain has no mail servers');
    break;
  case 'mailbox_not_found':
    console.log('Mailbox does not exist (deep mode)');
    break;
  case 'mailbox_full':
    console.log('Mailbox is full (deep mode)');
    break;
  case 'mailbox_disabled':
    console.log('Mailbox is disabled (deep mode)');
    break;
  case 'catch_all':
    console.log('Domain accepts all emails (deep mode)');
    break;
  case 'smtp_error':
    console.log('Could not connect to mail server (deep mode)');
    break;
}
```

## Common Use Cases

### Signup Form Validation

```typescript
import { validateEmail } from 'sniffmail';

async function validateSignupEmail(email: string): Promise<string | null> {
  const result = await validateEmail(email);

  if (!result.valid) {
    switch (result.reason) {
      case 'invalid_syntax':
        return 'Please enter a valid email address';
      case 'disposable':
        return 'Disposable email addresses are not allowed';
      case 'no_mx_records':
        return 'This email domain does not exist';
      default:
        return 'Please enter a valid email address';
    }
  }

  return null; // Valid
}
```

### Deep Verification for Critical Flows

```typescript
import { configure, validateEmail } from 'sniffmail';

// Configure once at app startup
configure({
  apiKey: process.env.SNIFFMAIL_API_KEY,
});

async function verifyEmailForInvoice(email: string): Promise<boolean> {
  const result = await validateEmail(email, { deep: true });

  if (!result.valid) {
    console.log(`Email ${email} failed: ${result.reason}`);
    return false;
  }

  if (result.smtp?.is_reachable === 'risky') {
    console.log(`Warning: ${email} is risky (catch-all or full inbox)`);
  }

  return true;
}
```

### Check Disposable Domain Only

For a quick synchronous check without network calls:

```typescript
import { isDisposableDomain } from 'sniffmail';

if (isDisposableDomain('tempmail.com')) {
  console.log('Blocked!');
}
```

## Validation Options

Customize the validation behavior with options:

```typescript
const result = await validateEmail('user@example.com', {
  deep: false,        // Enable SMTP verification (default: false)
  checkMx: true,      // Check MX records (default: true)
  useDeBounce: true,  // Use DeBounce API (default: true)
  timeout: 5,         // DNS timeout in seconds (default: 5)
});
```

## Error Handling

```typescript
import {
  validateEmail,
  ApiKeyNotConfiguredError,
  SniffmailError
} from 'sniffmail';

try {
  const result = await validateEmail('test@example.com', { deep: true });
} catch (error) {
  if (error instanceof ApiKeyNotConfiguredError) {
    console.error('API key not set. Get one at https://sniffmail.io');
  } else if (error instanceof SniffmailError) {
    if (error.statusCode === 403) {
      console.error('Usage limit exceeded. Upgrade at https://sniffmail.io');
    } else if (error.statusCode === 429) {
      console.error('Rate limited. Slow down requests.');
    }
  }
}
```
