## Overview

Sniffmail offers two validation modes that balance speed, cost, and accuracy:

| Mode | What It Checks | API Key Required | Cost | Speed | Use Case |
|------|----------------|------------------|------|-------|----------|
| **Quick Mode** | Syntax, disposable, MX | No | Free | &lt;100ms | Signup forms, contact forms |
| **Deep Mode** | Quick + SMTP verification | Yes | Paid | 1-3s | Payment flows, critical emails |

## Quick Mode (Default)

**Free, no API key needed, unlimited usage.**

Quick mode performs 4 validation checks:

<Steps>

<Step title="Syntax Validation">
Verifies email format is valid:
```typescript
const EMAIL_REGEX = /^[^\s@]+@[^\s@]+\.[^\s@]+$/;
```
</Step>

<Step title="Disposable Detection">
Checks against three blocklists:
- **GitHub blocklist** (26,000+ domains)
- **Scraped domains** (from temp mail sites)
- **Discovered domains** (found at runtime)
</Step>

<Step title="MX Record Lookup">
Verifies the domain has mail servers.
</Step>

<Step title="DeBounce API Check">
Real-time disposable detection via free API.
</Step>

</Steps>

### Usage

Quick mode is the default — no configuration needed:

```typescript
import { validateEmail } from 'sniffmail';

const result = await validateEmail('user@example.com');

if (!result.valid) {
  console.log(`Rejected: ${result.reason}`);
  // Possible reasons:
  // - 'invalid_syntax'
  // - 'disposable'
  // - 'no_mx_records'
}
```

### What Quick Mode Doesn't Catch

<Warning>
Quick mode **cannot detect**:
- Fake email addresses on real domains (`fakeperson123@gmail.com`)
- Typos in usernames (`johnn@gmail.com` when you meant `john@gmail.com`)
- Disabled or full mailboxes
- Catch-all domains (accept all addresses)

For these cases, use **Deep Mode**.
</Warning>

### Performance

- **Latency:** 50-100ms typical, 200ms worst-case
- **Throughput:** Unlimited (no rate limits)
- **Caching:** DNS results cached for 1 hour

```typescript

  concurrency: 10, // 10 parallel requests

// Completes in ~5-10 seconds
```

## Deep Mode (SMTP Verification)

**Requires API key. Verifies mailbox actually exists.**

Deep mode performs all Quick Mode checks **plus** SMTP verification:

<Steps>

<Step title="All Quick Mode Checks">
Syntax, disposable detection, MX lookup, DeBounce API.
</Step>

<Step title="Cache Lookup">
Checks if email was validated recently (avoids redundant API calls).
</Step>

<Step title="SMTP Connection">
Connects to the mail server to verify mailbox existence:
```typescript
// Powered by Sniffmail API
const apiResponse = await checkMailbox(email);
```
</Step>

</Steps>

### Setup

<Steps>

<Step title="Get API Key">
1. Sign up at [sniffmail.io](https://sniffmail.io)
2. Choose a plan (free tier: 500 verifications/month)
3. Copy your API key (starts with `sniff_`)
</Step>

<Step title="Configure SDK">
```typescript
import { configure } from 'sniffmail';

configure({
  apiKey: process.env.SNIFFMAIL_API_KEY,
});
```

Or set environment variable:
```bash
export SNIFFMAIL_API_KEY=sniff_xxxxx
```
</Step>

<Step title="Enable Deep Mode">
```typescript
const result = await validateEmail('user@gmail.com', { 
  deep: true 
});

if (result.smtp?.is_reachable === 'invalid') {
  console.log('Mailbox does not exist!');
}
```
</Step>

</Steps>

### SMTP Result Details

Deep mode returns detailed SMTP information:

```typescript
interface SmtpResult {
  is_reachable: 'safe' | 'risky' | 'invalid' | 'unknown';
  can_connect: boolean;      // Successfully connected to SMTP server
  is_deliverable: boolean;   // Mailbox accepts mail
  is_catch_all: boolean;     // Domain is catch-all (accepts all addresses)
}
```

**Example response:**

```typescript
const result = await validateEmail('john@gmail.com', { deep: true });

console.log(result);
// {
//   email: 'john@gmail.com',
//   valid: true,
//   reason: null,
//   disposable: false,
//   mx: true,
//   smtp: {
//     is_reachable: 'safe',
//     can_connect: true,
//     is_deliverable: true,
//     is_catch_all: false,
//   },
//   cached: false,
// }
```

## Choosing the Right Mode

### Use Quick Mode When:

<Check>**Signup forms** — Block disposable emails instantly</Check>
<Check>**Contact forms** — Prevent spam from temp mail services</Check>
<Check>**Newsletter signups** — Fast validation, no API key needed</Check>
<Check>**High volume** — Unlimited free usage</Check>
<Check>**Low-stakes validation** — Bounces won't cause major issues</Check>

### Use Deep Mode When:

<Check>**Payment flows** — Verify invoice/receipt emails before charging</Check>
<Check>**Critical notifications** — Ensure password resets reach users</Check>
<Check>**B2B signups** — Verify business emails are real</Check>
<Check>**Email list cleaning** — Remove invalid addresses before campaigns</Check>
<Check>**Two-factor auth** — Confirm email-based 2FA will work</Check>

## Example: Hybrid Approach

Use Quick Mode by default, Deep Mode for critical actions:

```typescript
import { validateEmail } from 'sniffmail';

// Signup: Quick mode
async function handleSignup(email: string) {
  const result = await validateEmail(email);
  
  if (!result.valid) {
    throw new Error(`Invalid email: ${result.reason}`);
  }
  
  // Allow signup
  return createUser(email);
}

// First purchase: Deep mode
async function handleFirstPurchase(email: string) {
  const result = await validateEmail(email, { deep: true });
  
  if (result.smtp?.is_reachable !== 'safe') {
    throw new Error('Please provide a valid email for your receipt');
  }
  
  return processPayment(email);
}
```

## Error Handling

### Missing API Key

```typescript
import { ApiKeyNotConfiguredError } from 'sniffmail';

try {
  await validateEmail('test@example.com', { deep: true });
} catch (error) {
  if (error instanceof ApiKeyNotConfiguredError) {
    console.error('Deep mode requires an API key from https://sniffmail.io');
  }
}
```

### Rate Limit Exceeded

```typescript
import { SniffmailError } from 'sniffmail';

try {
  await validateEmail('test@example.com', { deep: true });
} catch (error) {
  if (error instanceof SniffmailError && error.statusCode === 429) {
    console.error('Rate limited. Slow down requests.');
  }
}
```

### Usage Limit Exceeded

```typescript
try {
  await validateEmail('test@example.com', { deep: true });
} catch (error) {
  if (error instanceof SniffmailError && error.statusCode === 403) {
    console.error('Monthly quota exceeded. Upgrade at https://sniffmail.io');
  }
}
```

## Configuration Options

Customize validation behavior:

```typescript
const result = await validateEmail('user@example.com', {
  deep: false,        // Enable SMTP verification (default: false)
  checkMx: true,      // Check MX records (default: true)
  useDeBounce: true,  // Use DeBounce API (default: true)
  timeout: 5,         // DNS timeout in seconds (default: 5)
});
```

<Warning>
**Don't disable `checkMx` or `useDeBounce` unless you have a specific reason.** This reduces accuracy.
</Warning>
